A. Two Vessels

Time Limit: 1 second Memory Limit: 256 megabytes Submit

You are given two vessels, one with capacity $a$ and the other with capacity $b$. You are also given a third vessel with capacity $c$. Initially, the first vessel is full of water, and the other two are empty. You can perform the following operation:

Choose two vessels, $i$ and $j$. Pour water from vessel $i$ to vessel $j$ until either vessel $i$ becomes empty or vessel $j$ becomes full.

Input

The first line contains an integer $t$ ($1 \le t \le 1000$) — the number of test cases. The description of the test cases follows.

Each test case consists of a single line containing four integers $a, b, c, d$ ($1 \le a, b, c, d \le 10^9$).

Example

Input
2
5 3 2 4
5 3 2 1
                                    
Output
2
1
Sub ID Verdict Time Language

Editorial (Mock)

The key observation is that you only need to move water from the larger vessel to the smaller one. Each move transfers $c$ units of water. You need to find how many moves are required to equalize the water levels or reach the target.

Let $diff = |a - b|$. We want to reduce this difference to 0. Each move reduces the difference by $2 \times c$ (since $c$ is taken from one and added to the other). Therefore, the number of moves is $\lceil \frac{diff}{2c} \rceil$.


import math

def solve():
    a, b, c = map(int, input().split())
    
    diff = abs(a - b)
    
    moves = math.ceil(diff / (2 * c))
    print(moves)

t = int(input())
for _ in range(t):
    solve()
