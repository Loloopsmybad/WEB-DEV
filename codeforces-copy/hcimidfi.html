Input

The first line contains an integer 't' (1 <= t <= 1000) — the number of test cases. Each of the next 't' lines contains two integers 'a' and 'b' (1 <= a, b <= 10^9).

Output

For each test case, print a single integer — the sum of 'a' and 'b' on a new line.
